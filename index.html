Game + Media = Fun
Game production
從過去老舊的286電腦、Atari2600主機、任天堂紅白機、a'Can主機、Game&Watch
掌上機到後來的Online Game、Xbox，或是現今的手機遊戲時代裡，大多以遊戲創造IP。
而要將現有的IP(Intelligence Property)帶入遊戲中並賦予它生命及靈魂，除了熟知的遊戲企劃、
程式、美術、配音、製作人員外，方塊夫的團隊裡還有一群瘋狂的行銷人，將虛實整合的行銷創意帶入遊戲的世界；
讓遊戲不再只是單一的遊戲，而是發展成1+1大於3的遊戲， 將現有故事內容無限延伸。
除此之外，近期內也幫許多遊戲大廠OEM(Original Equipment Manufacturing)委託代工，
經過與多家日本遊戲大廠的交流更提升團隊人員美術的素質。
